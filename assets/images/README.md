DESTINATION IMAGES GO HERE
==========================

Drop the real destination and package images into this folder using the exact
file names listed in ../../image-manifest.md, for example:

  blk01.jpg   (Gargantua)
  hab01.jpg   (Pandora)
  neb01.jpg   (Pillars of Creation)
  pkg01.jpg   (The Guardian's Odyssey)
  ...

Until a real image exists, every card automatically falls back to
assets/example.jpg, so nothing ever looks broken.

You do NOT need to change any code — the file names are already referenced in
data/destinations.xml. Just add the images and refresh the page.
