ASSETS FOLDER
=============

Keep the following files in this folder for the site to look complete:

  logo.png         -> your brand logo (shown in the header)
  favicon.ico      -> browser tab icon
  Spaceflight.mp4  -> looping background video for the homepage hero
  example.jpg      -> placeholder image used by every card (a copy is included)
  images/          -> put the real destination images here later

NOTE: This zip only ships a placeholder "example.jpg" so the cards are not
empty. Your logo.png, favicon.ico and Spaceflight.mp4 were intentionally NOT
included so your originals are never overwritten. Just make sure they are here.

The homepage <video> tag points to "assets/Spaceflight.mp4" (capital S). If your
file uses a different name or capitalisation, either rename the file or update
the <source src="..."> line in index.html to match.
